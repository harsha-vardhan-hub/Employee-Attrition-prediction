🚀 Employee Attrition Prediction & Workforce Management System

📌 Overview

This project focuses on predicting employee attrition using Machine Learning and supporting workforce planning through interactive Power BI dashboards. Employee attrition is a critical issue that affects organizational performance, increases hiring costs, and impacts productivity.
The system uses historical HR data to identify patterns and predict which employees are likely to leave, enabling organizations to take proactive retention measures.

🎯 Objectives

Predict employee attrition using Machine Learning
Identify key factors influencing employee turnover
Provide actionable insights for HR decision-making
Visualize data using interactive Power BI dashboards

🧠 Machine Learning Approach

Algorithm Used: Random Forest Classifier
Problem Type: Binary Classification (Attrition: Yes/No)
Model trained on HR dataset with multiple employee features
The model analyzes relationships between features like:
Job Role
Job Satisfaction
Overtime
Monthly Income
Work Experience
and predicts whether an employee is likely to leave.

📊 Dataset Details

Dataset: HR Employee Attrition Dataset
Features: 30+ attributes including:

Work-Life Balance

The dataset contains structured employee data used for training and testing the model.
